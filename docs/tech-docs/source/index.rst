=================================================================
Coopr - Cluster Oriented Operations and Provisioning of Resources 
=================================================================

Coopr is a software system that enables end users to provision clusters based on templates and settings defined by
administrators. Loom exposes simple interfaces to administrators and allows easy customization of the properties of provisioned
clusters, including software services, hardware configurations and infrastructure providers.

This documentation describes what Coopr is, as well as how it works.
